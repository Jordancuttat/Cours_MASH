Ce ficher contient les éléments du premier exercice de création d'un schéma TEI : 
- le schéma lui-même, en format .rnc (RELAX NIG schema - compact syntax)
- la documentation à propos de ce schéma en .html
- le fichier d'enregistrement du schéma (myTEI.xml)
- le fichier pour tester ce schéma, repris du cours 6 : NussbaumerBaptiste_exo2.xml
- le fichier d'Oxygen en .xpr, qui est le projet réunissant tous ces éléments.
